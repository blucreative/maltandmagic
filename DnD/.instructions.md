# D&D Directory — SRD Rules Accuracy Requirements

When working on any file in the `DnD/` directory, the following rules are mandatory:

## Required Skill

Read and apply [.github/skills/srd-5.2.1-reference/SKILL.md](../../.github/skills/srd-5.2.1-reference/SKILL.md) for **all** work involving spells, class features, conditions, or mechanics.

## Mandatory SRD Validation

Before adding, editing, or describing any spell or class feature on a D&D character sheet:

1. Fetch the SRD source using the raw GitHub URLs in the skill.
2. Confirm the spell or feature exists in the SRD by name.
3. Confirm the spell is on the correct class's spell list.
4. Use only the exact casting time, range, components, duration, and effect from the SRD.

**Never invent spells or class features.** If content cannot be verified in the SRD 5.2.1, label it explicitly as homebrew.
